Class Entity

An Entity is every object you may have on your system. A character, a weapon, an skill, a map. Everything is an Entity. Entities are bag of Components, and Components describe how the data exists on the entities. Entities have an id field that can only be set once and will throw when you try to get one when no id is set. This set can be used to persist the entity on a database.
